# 影视资源搜索机器人 使用指南

## 基本用法

在群里发送以下格式即可搜索：

```
搜 关键词
```

例如：
- `搜 流浪地球`
- `搜索 庆余年`
- `搜三体`

默认搜索**夸克网盘**资源，每次返回最新的 **2 条**结果。

---

## 指定网盘

在关键词后面加上网盘名称，可以搜索指定网盘的资源：

```
搜 关键词 网盘名称
```

例如：
- `搜 流浪地球 百度` → 搜索百度网盘
- `搜 庆余年 阿里` → 搜索阿里云盘
- `搜 三体 迅雷` → 搜索迅雷网盘
- `搜 权力的游戏 115` → 搜索115网盘

不加网盘名称则默认搜索夸克网盘。

**重要提醒：**

- 链接是临时的，**收到后请尽快打开并转存到自己的网盘**
- 如果有提取码，打开链接时需要输入
- 不要相信网盘文件夹里的任何广告或引导信息

## 搜索技巧

1. **关键词尽量简短**：`搜 庆余年` 比 `搜 庆余年第二季全集1080P` 效果更好
2. **搜不到换个词试试**：有些资源可能用别名，比如 `搜 哈利波特` 搜不到可以试试 `搜 Harry Potter`
3. **耐心等待**：搜索需要几秒到几十秒不等，看到"正在搜索中"后请耐心等待
4. **避免重复搜索**：短时间内搜同一个关键词会直接返回缓存结果，不用担心重复搜索浪费资源
